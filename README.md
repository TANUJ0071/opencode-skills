## Installation

### Option 1: Using npx skills (Recommended)

```bash
# Install a specific skill
npx skills add TANUJ0071/opencode-skills@<skill-name> -g -y

# Example: Install php-expert
npx skills add TANUJ0071/opencode-skills@php-expert -g -y
```

### Option 2: Manual Installation

```bash
# Clone the repo
git clone https://github.com/TANUJ0071/opencode-skills.git ~/opencode-skills

# Copy specific skill to your skills folder
cp -r ~/opencode-skills/<skill-name> ~/.agents/skills/

# Or copy all skills
cp -r ~/opencode-skills/* ~/.agents/skills/
```
